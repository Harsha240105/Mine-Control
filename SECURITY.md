# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| >= 1.0.46 | :white_check_mark: |
| < 1.0.46  | :x:                |

## Reporting a Vulnerability

MineControl OS stores sensitive data locally including server configurations
and authentication tokens. If you discover a security vulnerability, please
report it privately by opening a GitHub Issue with the label `security`.

Do not disclose security vulnerabilities publicly until they have been
addressed by the maintainer.

## What to Include

- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested mitigation

## Response Time

You can expect an initial response within 48 hours.
